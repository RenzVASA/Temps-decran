# Temps d'écran — Filière CIEL

Projet de collecte et visualisation des habitudes numériques des élèves de la filière CIEL.
Données collectées auprès de 13 élèves de 1ère CIEL sur une semaine complète.

## Données

- Classe : 1ère CIEL
- Élèves : 13
- Période : 1 semaine
- Moyenne temps d'écran : 69h/semaine par élève
